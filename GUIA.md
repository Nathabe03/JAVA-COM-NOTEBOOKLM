## Conceitos Fundamentais e Ecossistema
Portabilidade: O conceito "Write Once, Run Anywhere" é possível graças ao Bytecode e à JVM.

Gestão de Memória: O Garbage Collector (Coletor de Lixo) automatiza a limpeza de objetos não utilizados, trazendo robustez ao sistema.

Ferramentas: Utilizamos o javac para compilar e IDEas como NetBeans, Eclipse ou IntelliJ para agilizar o desenvolvimento.

## Pilares de POO (Programação Orientada a Objetos)
Abstração: Representar conceitos do mundo real no código.

Encapsulamento: Proteger os dados através de métodos Getters e Setters.

Herança (Inheritance): Reutilizar características de uma classe em outra.

Polimorfismo: Capacidade de um objeto ser referenciado de várias formas.

## Sintaxe e Estruturas Básicas
Tipos Primitivos: int, double, boolean, char.

Controle de Fluxo: if/else, switch case.

Laços: for, while, do-while.

Estruturas Compostas: Arrays e Collections.

## Glossário de Termos
Maven: Ferramenta para gerenciamento de dependências.

JUnit: Framework utilizado para criação de testes unitários.

Spring Boot: Framework focado em Desenvolvimento Web e sistemas empresariais.
